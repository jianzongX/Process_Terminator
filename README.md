# 进程终结者 Process_Terminator
可以结束带父进程的Windows可执行文件程序。
Can terminate Windows executable programs along with their parent processes.
## 一款针对酷信多媒体课堂和红蜘蛛多媒体课堂的进程结束软件 软件使用C++编写
